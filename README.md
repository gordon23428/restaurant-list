# 餐廳清單
## Features/功能
裡面有需多餐廳選擇，可以搜尋自己喜歡的餐廳，點開餐廳會有詳細資訊提供參考

## Prerequisites/環境建置
+ Node.js
+ Express
+ Express handlebars
+ Nodemon

## Installation and execution/安裝與執行步驟
### Clone
```
git clone https://github.com/gordon/restaurant-list
cd restaurant-list
npm install
```
### Run
```
npm run start
```
如果成功會看到
```
Express server is running on http://localhost:3000
```
可至 http://localhost:4000 查看
